# 📊 HR Analytics Project (Excel + SQL + Power BI)

## 📌 Project Overview

This project demonstrates an end-to-end data analytics workflow using **Excel, SQL, and Power BI**.

The goal is to analyze employee data and generate meaningful business insights related to salary distribution, employee performance, and workforce trends.

---

## 🛠 Tools & Technologies

* Microsoft Excel → Data Cleaning & Preparation
* SQL → Data Analysis & Querying
* Power BI → Data Visualization & Dashboard

---

## 🔄 Project Workflow

### 1️⃣ Excel (Data Cleaning)

* Removed inconsistencies and formatted data
* Created new columns:

  * Salary Category (High / Medium / Low)
  * Experience Level (Senior / Junior)
* Ensured clean and structured dataset

---

### 2️⃣ SQL (Data Analysis)

* Imported cleaned data into SQL database
* Performed analysis using queries:

  * Total Employees
  * Average Salary by Department
  * Top Performing Employees
  * Location-wise Distribution
* Identified key business insights

---

### 3️⃣ Power BI (Dashboard)

* Built an interactive dashboard with:

  * KPI Cards (Total Employees, Avg Salary, Avg Rating)
  * Bar Charts (Department, Location, Experience)
  * Pie Chart (Gender Distribution)
  * Top Performers Table (Top N Filter)
* Added slicers for dynamic filtering

---

## 📊 Key Insights

* Finance department has the highest average salary
* Senior employees earn more than junior employees
* Hyderabad has the highest employee count
* Top performers are mainly from Finance and IT
